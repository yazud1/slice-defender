# Slice Defender — 3D Interactive Game

Slice Defender is a 3D interactive game where the player defends against incoming projectiles
by slicing them with their hand detected via webcam.

This project combines real-time 3D rendering and computer vision.

## Features
- 3D scene rendered with OpenGL (textures, lighting, animations)
- Real-time hand detection using OpenCV and webcam
- Interactive gameplay with object slicing and fragmentation
- Qt-based graphical user interface

## Tech Stack
- C++
- Qt / Qt Creator
- OpenGL
- OpenCV

## Project context
Academic project (2024–2025) focused on multimedia development, real-time interaction,
and computer vision.
